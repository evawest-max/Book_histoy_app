About this app

This app was built by Ebunoluwa Giobari Akinwumi a Nigerian from Ondo state.
He is a web developer and also a mechanical engineer. The app was built to store and list the books that have benn read by the user.

How to use the app

The Add button is use to add books you have read to the database
The Clear database button is use to remove the books you have added to the database.
The Post button is used to publishe all the books in the database.
